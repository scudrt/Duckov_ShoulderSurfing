# 模组简介
逃离鸭科夫 第三人称越肩视角模组
让逃离鸭科夫成为越肩视角的射击游戏

# 特性列表
1. 将游戏相机转为第三人称越肩视角相机，允许调节相机偏移、FOV以及渲染距离等参数

2. 支持360度自由瞄准、Y轴反转以及切换左右肩等操作

3. 采用平滑的相机碰撞，以及支持角色渐隐防遮挡，给予你更好的视角体验

4. 为越肩视角优化的UI显示设计，包括方向正确的声纹、动态血条、体力条以及透视开关等效果

5. 越肩视角下的灵敏度与原版灵敏度挂钩，并且有专门的系数调节

6. 支持不同倍镜的放大倍率，放大倍率与原版倍镜和枪械挂钩

7. 源自原版的后坐力机制，不过暂时仅支持垂直后坐力

8. 采用原版地图的简易小地图，可以自由调节位置与大小，并且可以跟随玩家视角旋转

9. 支持自定义快捷键，包括视角开关、小地图开关以及切换肩位等操作的快捷键

10. 其他小特性，在MOD设置中可以找到(需要MODSetting作为前置)

# 如何构建
1. clone本源码

2. 参考鸭科夫官方文档，构建MOD编译环境（或者用VS Code打开项目文件夹，它会提示并帮助你构建本项目所需的环境）

3. 修改.csproj文件，将其中的`DuckovPath`一项改为你本地的游戏路径

4. 使用Release版本，开始构建

5. 构建成功后，游戏目录下Mods文件夹将会自动出现构建好的越肩MOD。打开游戏，可以在MODS菜单中找到这个MOD

6. 本MOD设置功能依赖`MODSetting`这个模组，没有该模组将无法使用MOD设置功能
